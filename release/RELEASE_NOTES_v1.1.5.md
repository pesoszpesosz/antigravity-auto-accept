## Antigravity Auto Accept v1.1.5

### Included assets

- `antigravity-auto-accept-1.1.5.vsix`
- `antigravity-auto-accept-1.1.5.vsix.sha256`
- `control-panel.png`

### What changed

- Added a read-only `Copy Launcher Steps` command and control-panel button.
- Added a read-only `Copy Manual Launch Command` command and control-panel button.
- Kept the new copy actions isolated to existing launcher/setup text so runtime behavior stays unchanged.

### Install

1. Download `antigravity-auto-accept-1.1.5.vsix` from this release.
2. In Antigravity, run: `Extensions: Install from VSIX...`
3. Select the VSIX and reload.
4. Open `Auto Accept Panel`.
5. Use `Copy Launcher Steps` or `Copy Manual Command` whenever you need to share or reuse the current setup instructions.
