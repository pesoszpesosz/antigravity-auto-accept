## Antigravity Auto Accept v1.0.8

### Included assets

- `antigravity-auto-accept-1.0.8.vsix`
- `antigravity-auto-accept-1.0.8.vsix.sha256`

### What changed

- Made the control panel entry point the first thing explained in the packaged README.
- Added explicit `v1.0.8` download links so the extension page no longer points users at an older build.
- Kept the free-build note, Open VSX rating request, and GitHub feedback request at the top.
- Preserved the screenshot and control-panel setup guidance in the public docs.

### Install

1. Download `antigravity-auto-accept-1.0.8.vsix` from this release.
2. In Antigravity, run: `Extensions: Install from VSIX...`
3. Select the VSIX and reload.
4. Open `Auto Accept Panel` from the bottom-right status area.
