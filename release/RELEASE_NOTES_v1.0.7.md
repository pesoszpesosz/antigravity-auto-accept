## Antigravity Auto Accept v1.0.7

### Included assets

- `antigravity-auto-accept-1.0.7.vsix`
- `antigravity-auto-accept-1.0.7.vsix.sha256`

### What changed

- Added the new control panel documentation and screenshot to the public project pages.
- Centered setup on the control panel, configurable CDP port, and save-anywhere launcher flow.
- Fixed the Windows launcher path to match the working Antigravity shortcut format used on the maintainer machine.
- Improved Windows runtime detection for the selected CDP port.

### Install

1. Download `antigravity-auto-accept-1.0.7.vsix` from this release.
2. In Antigravity, run: `Extensions: Install from VSIX...`
3. Select the VSIX and reload.
4. Open `Antigravity Auto Accept: Open Control Panel`.
5. Save the launcher you want to use and open Antigravity through it.

Detailed install guide: `INSTALL.md`.
